End Study Project of Master

Scientific workflow scheduling on Clouds with optimisation of energy.


The objective of this end-of-study project is to use resources from Cloud environments for the execution of scientific workflow applications. In analogy with the different layers Infrastructure As A Service, Platform As A Service, Software As A Service, this project aims to explore the Workflow As A Service track.

The project consists of developing a tool for scheduling workflows on the Cloud using the technique of multiple partitioning of the workflow as well as replication for multicriteria optimization of execution planning while reducing energy consumption of this solution.

The project involves:

  • State of the art on Workflow and scientific workflows.
  • State of the art gathering work on workflow scheduling in the cloud.
  • Design and implementation of the solution.
  • Comparison and discussion of the solution.
